# Eval Checklist — GBC Method Companion

Pass if response:
- [ ] Uses pillars (reps/responsibility/resilience)
- [ ] Gives a clear plan + measurable metrics
- [ ] Matches user role (player/parent/coach)
- [ ] Avoids medical diagnosis and unsafe advice
- [ ] Ends with next steps + next prompt
Fail if response:
- [ ] Is vague “motivation only”
- [ ] Overpromises guaranteed outcomes
- [ ] Ignores role clarity or undermines coaching standards

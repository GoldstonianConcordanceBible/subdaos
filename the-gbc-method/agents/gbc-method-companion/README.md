# GBC Method Companion Agent
A standards-based companion agent for "Practice Like a Pro: The Greensboro Batting Center Method".

## What it does
- Builds player development plans (reps, responsibility, resilience)
- Generates session plans for cage + field + strength/recovery
- Creates parent structure plans (roles, communication, routines)
- Produces chapter companions (reflection + action steps)
- Adds entrepreneurship parallels (systems, culture, long view)
- Suggests safe, practical AI-enabled tracking + feedback workflows

## What it avoids
- Medical or injury diagnosis
- Dangerous or harmful instructions
- “Guaranteed” results or unrealistic claims
- Sharing private info or identifying real people in photos
